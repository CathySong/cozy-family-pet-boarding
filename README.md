# Cozy Family Pet Boarding Website

A responsive, modern website for a family-run pet boarding service in Longhill, NJ.

## Features

- **Responsive Design**: Works on all devices from mobile to desktop
- **Modern UI**: Clean, warm design with pet-friendly colors
- **Complete Information**: Services, pricing, gallery, and contact details
- **Booking Integration**: Calendly integration for easy scheduling
- **Photo Gallery**: Showcase of happy pets who have stayed

## File Structure

- `cozy-family-index.html` - Main website HTML file with embedded CSS and images

## Website Details

The website includes:
- Home page with hero section and key features
- Services overview with detailed pricing
- Photo gallery of happy pets
- Family information and trust indicators
- Contact information and booking system
- Mobile-responsive design

## Technologies Used

- HTML5
- CSS3 (with custom properties and modern layout)
- Google Fonts (Nunito and Baloo 2)
- Embedded images (base64 encoded for portability)
- Calendly integration for booking

## How to Use

Simply open `cozy-family-index.html` in any modern web browser to view the website.

## Deployment

The website is ready to be deployed to any web hosting service. The single HTML file contains all necessary CSS and images embedded within it.

## Customization

To customize the website:
1. Edit the HTML file directly
2. Update contact information in the contact section
3. Replace base64-encoded images with your own
4. Update Calendly links for your booking system
5. Modify pricing and service details as needed
